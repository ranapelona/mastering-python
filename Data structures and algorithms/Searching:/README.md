## Featured: Binary Search ("The moving organism")
A high _ efficiency search algorithm that halves the search with every repeat of the loop like a heartbeat.

### 🧠 The logic map 
Before coding i mapped out the algorithm's "heartbeat" to enure precision. I treated the "low" and "high" pointers as dynamic boundaries
that close in on the target.

![Binary Search Logic Map](./assets/binary_search_logic_map.jpg

### What i learned

***Midpoint Exclusion*** I used "low = mid +1" and "high = mid -1" to make sure that the boundaries close in on the search space.
***Termination*** This precision prevents the algorithm from getting paralyzed in an infinite loop once it identifies the target coordinates.
***Efiiciency*** This approach achieves $O(log n)$ time complexity, making it exponentially faster than a linear search.
